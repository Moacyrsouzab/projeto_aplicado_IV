# ANÁLISE E MODELAGEM PREDITIVA DE SÉRIE TEMPORAL - AIR PASSENGERS ✈
> **Projeto Aplicado IV - Ciência de Dados** > Universidade Presbiteriana Mackenzie - 2026

## 📌 Sobre o Projeto

Este projeto está inserido na área de Ciência de Dados e Analytics, com foco específico em análise e modelagem de séries temporais. O trabalho utiliza como objeto de estudo o conjunto de dados Air Passengers, disponível na plataforma Kaggle, que contém o registro mensal do número total de passageiros de uma companhia aérea internacional no período de 1949 a 1960. 

O problema central abordado neste projeto consiste em **como modelar e prever o comportamento futuro da demanda de passageiros a partir de padrões históricos, considerando tendência, sazonalidade e crescimento ao longo do tempo.**
